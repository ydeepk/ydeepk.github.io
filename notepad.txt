Local Variables (Method Scope)
Declared inside methods or constructors.
Scope limited to the block in which they are declared.
Must be explicitly initialized before use.
Stored in stack memory.
Destroyed after method execution completes.
Method Parameters
Parameters are local variables to the method.
Scope is limited to the method body.
Can shadow instance variables.
this is used to differentiate instance variables from parameters with the same name.
Block Scope
Variables declared inside { } are accessible only within that block.
Applies to if, for, while, and initializer blocks.
Local variables cannot be redeclared in overlapping blocks.
